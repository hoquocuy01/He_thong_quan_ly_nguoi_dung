## 2. Công nghệ sử dụng
- Ngôn ngữ: Python (Flask Framework)
- Cơ sở dữ liệu: MySQL (XAMPP)
- Giao diện: HTML5, CSS3 (Modern Glassmorphism)

## 3. Hướng dẫn cài đặt & thực nghiệm
1. Tải mã nguồn hoặc dùng lệnh: `git clone [https://github.com/hoquocuy01/He_thong_quan_ly_nguoi_dung]`
2. Cài đặt môi trường: `pip install -r requirements.txt`
3. Import file `database.sql` vào MySQL thông qua phpMyAdmin.
4. Chạy ứng dụng: `python app.py`

## 4. Tài khoản thử nghiệm
- **Admin:** `admin` / mật khẩu: `123456`
- **User thường:** (Bạn có thể tự đăng ký trên giao diện)

## 5. Các tính năng nổi bật
- Giao diện kính mờ (Glassmorphism) đồng bộ.
- Admin có quyền Khóa/Mở khóa người dùng tức thì.
- Chế độ bảo vệ tài khoản đang đăng nhập.
